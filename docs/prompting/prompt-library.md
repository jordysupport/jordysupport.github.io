# Prompt library

Copy, fill in the `[brackets]`, paste. Every prompt ends with a plan-first line so nothing runs away from you.

## Summarize anything

```text title="Copy this"
Summarize the material below for [audience] in under [150] words.
Plain English, no hype. If something important is unclear or
missing, say so rather than smoothing over it.

[paste material]
```

## Improve my draft (without rewriting my voice)

```text title="Copy this"
Improve the draft below: fix grammar, tighten wording, flag
anything confusing. Keep my voice and my meaning — do not add
new points. Show the result, then list what you changed.

[paste draft]
```

## Compare options

```text title="Copy this"
Compare [option A] and [option B] for [my situation]. Give me a
short table of the differences that matter to me, then a plain
recommendation with the main tradeoff. Say what you don't know.
```

## Explain something to me

```text title="Copy this"
Explain [topic] like I'm smart but new to this. Short paragraphs,
no jargon without a plain-word translation, one concrete example.
End with the two things most people get wrong about it.
```

## Turn chaos into a plan

```text title="Copy this"
Below is a messy brain-dump. Turn it into: goals, next actions
(smallest first), open questions, and things to ignore for now.
Ask me one clarifying question if you need to.

[paste brain-dump]
```

## Write a first draft

```text title="Copy this"
Draft a [email / post / page] about [topic] for [audience].
Shape: [length, tone]. Use only the facts below — don't invent
any. This is a draft for my review.

Facts: [paste facts]
```

## Check work (yours or the AI's)

```text title="Copy this"
Act as a careful reviewer. Check the text below for: factual
claims that need a source, contradictions, unclear sentences,
and anything that could embarrass me. List problems by line —
don't rewrite yet.

[paste text]
```
