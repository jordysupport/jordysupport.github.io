# The basics

An [**agent**](../resources/glossary.md#agent) is an AI that can *do* things — read files, write drafts, run steps — not just chat. This section explains how they work, in plain words, so you can supervise one with confidence.

## The six pieces of every agent

1. **Instructions** — the job and the rules you give it.
2. **Context** — what it can currently see (your message, files, notes).
3. **Tools** — what it's allowed to use (files, web, apps).
4. **Actions** — what it actually does.
5. **Checks** — how you (or it) verify the work.
6. **Stop points** — where it pauses and asks you.

Every problem you'll ever have with an agent traces back to one of those six. That makes fixing things much less mysterious.
